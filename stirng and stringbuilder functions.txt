1. to convert character to lowercase -> Character.toLowerCase(char) 

2. to check character is alphanumeric or not -> Character.isLetterOrDigit(char ch)

3. "String" functions ->

charAt(int index) ->	Get char at index
	
substring(start) ->  Substring from start to end		
substring(start, end) -> From start to end-1

indexOf(char/str) -> Index of first occurrence

lastIndexOf(char/str) -> Last occurrence
	
contains(CharSequence)-> Check if contains substring

startsWith(str) ->Starts with string?
	
endsWith(str)->Ends with string?

// Modify or replace

| Method               | Description                    | Example                              |
| -------------------- | ------------------------------ | ------------------------------------ |
| `replace(old, new)`  | Replace all occurrences        | `"aabb".replace("a","c")` → `"ccbb"` |
| `toLowerCase()`      | Convert to lowercase           | `"HELLO".toLowerCase()`              |
| `toUpperCase()`      | Convert to uppercase           | `"hello".toUpperCase()`              |
| `trim()`             | Remove leading/trailing spaces | `"  hi  ".trim()` → `"hi"`           |
| `split(regex)`       | Split by delimiter             | `"a,b,c".split(",")`                 |
| `replaceFirst(a, b)` | Replace first match            | `"aabb".replaceFirst("a", "z")`      |


// compare or check

| Method                  | Description                 | Example                         |
| ----------------------- | --------------------------- | ------------------------------- |
| `equals(str)`           | Case-sensitive comparison   | `"abc".equals("abc")` → `true`  |
| `equalsIgnoreCase(str)` | Case-insensitive comparison | `"abc".equalsIgnoreCase("ABC")` |
| `compareTo(str)`        | Lexicographical comparison  | `"abc".compareTo("abd")` → `-1` |
| `isEmpty()`             | Check if string is empty    | `"".isEmpty()` → `true`         |
| `length()`              | Length of string            | `"hello".length()` → `5`        |

4. StiringBuilder class functions

| Method                | Description                    | Example              |
| --------------------- | ------------------------------ | -------------------- |
| `append(str)`         | Add to end                     | `sb.append("abc")`   |
| `insert(index, str)`  | Insert at index                | `sb.insert(2, "z")`  |
| `delete(start, end)`  | Delete from `start` to `end-1` | `sb.delete(1, 3)`    |
| `deleteCharAt(index)` | Delete character at index      | `sb.deleteCharAt(0)` |

| Method                     | Description                   | Example                  |
| -------------------------- | ----------------------------- | ------------------------ |
| `replace(start, end, str)` | Replace range with new string | `sb.replace(0, 2, "xy")` |
| `reverse()`                | Reverse the whole string      | `sb.reverse()`           |

| Method             | Description           | Example               |
| ------------------ | --------------------- | --------------------- |
| `indexOf(str)`     | First occurrence      | `sb.indexOf("a")`     |
| `lastIndexOf(str)` | Last occurrence       | `sb.lastIndexOf("a")` |
| `length()`         | Get current length    | `sb.length()`         |
| `charAt(index)`    | Character at position | `sb.charAt(1)`        |

| Method                   | Description          | Example                |
| ------------------------ | -------------------- | ---------------------- |
| `setCharAt(index, char)` | Modify char at index | `sb.setCharAt(0, 'z')` |
| `toString()`             | Convert to `String`  | `sb.toString()`        |
| `capacity()`             | Buffer capacity      | `sb.capacity()`        |
